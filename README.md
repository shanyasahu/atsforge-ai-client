ATSForge AI – Frontend

ATSForge AI is an AI-powered resume optimization platform that helps job seekers craft ATS-friendly resumes, analyze job descriptions, identify skill gaps, and prepare for job interviews.

This repository contains the frontend client application built with React, Vite, and TailwindCSS.

Features
AI-powered Resume Optimization
ATS Compatibility Analysis
Job Description Parsing
Skill Gap Detection
Interview Preparation Insights
Fast and modern UI using React + Vite
Client side routing with React Router
API communication via Axios
